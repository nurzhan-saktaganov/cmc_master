Проверено на GNU Octave 3.8.1

poisson.m - прямой метод решения СЛАУ,
	принимает аргументы
		f - правая часть
		g - краевое условие
		m - количество отрезков

Задание А, нулевые краевые условия
Тестовые функции
	f_a.m - правая часть
	g_a.m - краевое условие, нулевая функция
	u_a.m - аналитическое решение
Тест с выводом графика
	task_a.m

Задание Б
Тестовые функции
	f_b.m - правая часть
	g_b.m - краевое условие
	u_b.m - аналитическое решение
Тест с выводом графика
	task_b.m

poisson_jac.m - итерационный метод Якоби решения СЛАУ,
	принимает аргументы
		f - правая часть
		g - краевое условие
		m - количество отрезков
		eps - точность
